# bendingStress
# Unsymmetric Bending Visualizer

Interactive Python tool for visualizing:

- stress distribution in a rectangular beam cross-section
- bending moment components
- 3D deformed beam shape under unsymmetric bending

## Features

- 2D stress map with neutral axis
- corner stress labels
- moment vector diagram
- 3D beam deformation view
- interactive sliders for angle, width, height, beam length, and deformation scale
